Announcing Frinj, a practical unit of measure calculator DSL for Clojure

2012-03-03 clojure frinj
I am proud to announce a new Clojure project called “Frinj”. Frinj is a practical unit-of-measure calculator DSL for Clojure. Key features; Tracks units of measure through all calculations allowing you to mix units of measure transparently Comes with a huge database of units and conversion factors Inspired by the Frink project Tries to combine Frink’s fluent calculation style with idiomatic Clojure Full source code available on github.
